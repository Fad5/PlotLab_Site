Инструкция по использованию:

1. Подготовка файлов

Требования к Excel-файлу:

Файл должен содержать следующие обязательные столбцы:

Название столбца Описание Формат
Образец Идентификатор образца Число (1, 2, 3...)
Высота Высота образца Числовое значение (мм)
Ширина Ширина образца Числовое значение (мм)
Длина Длина образца Числовое значение (мм)
Масса Базовая масса образца Числовое значение (кг)
Массы пригрузов Столбцы с числовыми названиями (2, 5, 10 и т.д.) Дополнительные параметры для каждого пригруза

Требования к архиву с данными:

  • Поддерживаемые форматы: .zip или .rar
  • Архив должен содержать CSV файлы с результатами измерений
  • Имена файлов должны соответствовать шаблону: "образец_масса.csv" (например: "1_2.csv", "1_5.csv")
  • CSV файлы должны содержать данные в формате: частота, амплитуда
  • Рекомендуется не более 15 образцов в одном архиве
Пример структуры Excel файла:
Образец Высота Ширина Длина Масса 2 5 10
1 10.0 100.0 100.0 120.0 9.0 8.0 7.0
2 10.0 100.0 100.0 125.0 9.0 8.0 7.0

2. Процесс работы

1
Загрузка файлов через форму

На странице анализа:

  1. Выберите Excel файл с параметрами образцов
  2. Выберите архив с данными измерений
  3. Нажмите кнопку "Сгенерировать отчет"
2
Обработка файлов системой

Система выполняет следующие действия:

  • Проверяет наличие всех обязательных столбцов в Excel-файле
  • Сопоставляет CSV файлы с параметрами образцов
  • Анализирует амплитудно-частотные характеристики
  • Вычисляет динамический модуль упругости и коэффициенты потерь
  • Строит сравнительные графики
3
Формирование результата

После обработки:

  • Создается подробный отчет в формате Word (.docx)
  • Формируются таблицы с результатами испытаний
  • Строятся графики амплитудно-частотных характеристик
  • Создаются сравнительные графики разных образцов
  • Отчет автоматически загружается на ваш компьютер

3. Содержание отчета

  • Геометрические характеристики образцов - таблица с параметрами
  • Результаты испытаний - подробные таблицы для каждого образца
  • Средние значения характеристик - с коэффициентами вариации
  • Графики зависимостей - динамический модуль упругости от давления
  • Сравнительные графики - АЧХ и эффективности виброизоляции
  • Графики всех образцов - индивидуальные характеристики

4. Ограничения и рекомендации

  • Максимальный размер архива: 50 МБ
  • Максимальное количество образцов: 15
  • Максимальное количество пригрузов на образец: 10
  • Для точных результатов убедитесь в правильности измерений
  • Избегайте специальных символов в именах файлов
  • Время обработки зависит от количества данных (1-5 минут)
