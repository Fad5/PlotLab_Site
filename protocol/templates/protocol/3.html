Скачать шаблоны

Используйте стандартные шаблоны для подготовки данных:

Скачать шаблон (.docx) Шаблон Excel (.xlsx)

Инструкция по использованию:

1. Подготовка файлов

Требования к Excel-файлу:

Файл должен содержать следующие обязательные столбцы:

Название столбца Описание Формат
Образец Идентификатор образца Текст (будет использован в имени файла)
Ширина Ширина образца Числовое значение (мм)
Длина Длина образца Числовое значение (мм)
Высота Высота образца Числовое значение (мм)
Масса Масса образца Числовое значение (г)
Номер протокола Номер протокола Текст или число
Дата Дата проведения испытаний Дата в любом формате

Требования к шаблону DOCX:

  • Должен быть создан в Microsoft Word или совместимом редакторе
  • Должен содержать поля для подстановки в формате: {{ Образец }}, {{ Ширина }} и т.д.
  • Поддерживаются все стандартные элементы Word (таблицы, изображения, форматирование)
Пример структуры шаблона:
ПРОТОКОЛ №{{ Номер_протокола }}
Дата: {{ Дата }}

Наименование образца: {{ Образец }}
Параметры:
- Ширина: {{ Ширина }} мм
- Длина: {{ Длина }} мм
- Высота: {{ Высота }} мм
- Масса: {{ Масса }} г

2. Процесс работы

1
Загрузка файлов через форму

На странице генерации:

  1. Выберите файл шаблона (.docx)
  2. Выберите файл с данными (.xlsx или .xls)
  3. Укажите диапазоны процентных значений сжатия
  4. Выберите тип испытуемого образца
  5. Нажмите кнопку "Сгенерировать протоколы"
2
Обработка файлов системой

Система выполняет следующие действия:

  • Проверяет наличие всех обязательных столбцов в Excel-файле
  • Читает данные из каждой строки Excel-файла
  • Для каждой строки создает отдельный протокол, подставляя данные в шаблон
  • Применяет указанные процентные диапазоны сжатия
3
Формирование результата

После обработки всех строк:

  • Все сгенерированные протоколы упаковываются в ZIP-архив
  • Архив автоматически загружается на ваш компьютер
  • Имена файлов в архиве: Протокол_[Образец].docx

3. Пример использования

Пример Excel-файла:
Образец Ширина Длина Высота Масса Номер протокола Дата
sample_1 10.2 15.5 5.1 25.4 2023-001 01.01.2023
sample_2 9.8 14.9 4.9 24.1 2023-002 02.01.2023
Пример процентных значений сжатия:
Тип сжатия Минимальное значение Максимальное значение
10% сжатие 0.05 0.15
20% сжатие 0.15 0.25
40% сжатие 0.35 0.45
Результат:

Будет создан ZIP-архив, содержащий два файла:

  • Протокол_sample_1.docx
  • Протокол_sample_2.docx

4. Ограничения и рекомендации

  • Максимальное количество строк в Excel-файле: 100
  • Размер шаблона DOCX не должен превышать 5 МБ
  • Размер Excel-файла не должен превышать 10 МБ
  • Для специальных форматов дат используйте предварительное форматирование в Excel
  • Избегайте специальных символов в именах образцов
  • Процентные значения должны быть положительными числами
  • Максимальное значение не может быть меньше минимального
  • Время обработки зависит от количества данных (1-3 минуты)

5. Типы образцов

Прямоугольная пластина:

  • Имеет разные значения ширины и длины
  • Подходит для стандартных прямоугольных образцов
  • Расчеты производятся с учетом разных размеров по осям

Квадратная пластина:

  • Ширина и длина равны
  • Подходит для квадратных образцов
  • Упрощенные расчеты за счет симметрии
