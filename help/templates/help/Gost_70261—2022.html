Определение удельной статической жесткости

  1. Определяем тип подшпальной прокладки.
  2. Смотрим на геометрические характеристики образца.
  3. Вычеслям нагрузку для испытания, по таблице. Дана нагрузка для образца размером 250х250.
    Если например у нас образец 100х100 тип 4, значит получаем коэффициент 250 / 100 = 2.5
    F_min = 437 / 2.5 = 175 H,
    F_max = 9375 / 2.5 = 3750 H,
    F_1 = = 625 / 2.5 = 250 H,
    F_2 = 3750 / 2.5 = 1500 H
    Если образец имеет неправельную форму находим среднее значение
  4. Проводят пятикратное сжатие в диапазоне нагрузок, со скоростью нагружения (37,5 ± 2) кН/мин.
